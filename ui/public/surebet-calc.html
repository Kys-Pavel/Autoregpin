Калькулятор вилок

Вставьте JSON с суребетами выше и нажмите «Загрузить». Калькулятор автоматически распределит сумму по плечам для оптимальной доходности.

📋 Что нужно иметь для проставки ставок на Pinnacle

eventId
1626892236
ID события Pinnacle. Нужен для построения URL ставки или API запроса. Уже есть в JSON.
betType
HDP / MONEYLINE / TOTALS
Тип ставки: HDP = гандикап, MONEYLINE = победитель, TOTALS = тотал. Уже есть в JSON.
period
"0" = матч, "1" = 1-й тайм
На какой период ставка. Уже есть в JSON.
side (HOME / AWAY / OVER / UNDER)
HOME или AWAY
На какое плечо ставить. Выбирается из prices в JSON. Уже есть в JSON.
handicap / hdp / altLineId
"0", "-1.5", "+0.5"
Значение гандикапа. Для MONEYLINE = 0. Уже есть в JSON.
price (odds)
1.952 / 1.8
Коэффициент, по которому ставим. Нужно 100% совпадение с текущим на сайте — иначе ставка отклоняется. Уже есть в JSON.
stake (сумма ставки)
50.00 USDT
Сумма ставки рассчитывается калькулятором. Генерируется калькулятором.
⚠️ username / password
loginId + password из accounts.json
Логин и пароль аккаунта Pinnacle. Нужно добавить в скрипт ставки. Берётся из data/accounts.json.
⚠️ mirror URL (зеркало)
https://thundercrest65.xyz
Ссылка на зеркало где авторизован аккаунт. Каждый аккаунт авторизован на своём зеркале. Нужно добавить в скрипт ставки.
⚠️ sessionToken / cookies
session cookie после логина
Pinnacle API требует авторизацию. Puppeteer должен залогиниться и держать сессию. Нужно реализовать.
⚠️ betslipId / uniqueRequestId
UUID генерируется на каждую ставку
Pinnacle API может требовать уникальный идентификатор на каждый бетслип. Генерируется при POST /bets.
deposit balance (баланс)
USDT ERC20
Баланс уже пополнен через модуль depositor.js. Ставки ставятся от этого баланса. Уже реализовано.
